TP3 Projet de gestion de forfaits de voyage


Partie 1 : Front-end avec Vue.js, Vite, Tailwind CSS et Axios

Dans cette première partie, le projet se concentre sur le développement du front-end avec Vue.js, Vite, Tailwind CSS et Axios.

Vue.js est utilisé pour créer l'interface et gérer la logique de l'application.

Vite est utilisé comme bundler pour un développement rapide et optimisé.

Tailwind CSS facilite la création de l'interface utilisateur avec des classes utilitaires.

Axios est utilisé pour faire des requêtes HTTP aux API exposées par le back-end, permettant de récupérer et afficher dynamiquement les données des forfaits.


Partie 2 : Back-end avec Node.js, Express et une base de données SQL

La deuxième partie du projet est dédiée au développement du back-end avec Node.js, Express, et une base de données SQL.

Création des routes API avec Express :

Une fois les données en base de données, je crée des routes API pour permettre au front-end de récupérer, afficher, ajouter ou modifier des forfaits.

J'utilise Sequelize pour effectuer des opérations CRUD (Create, Read, Update, Delete) sur la base de données.
